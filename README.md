# GallBladder-Disease-Classification
CNN-based gallbladder disease classification using 224Ã—224 images with memory-optimized training, mixed precision, and class weighting. Model achieved high accuracy on train/val/test sets. Results analyzed via Grad-CAM, confusion matrix, sample predictions, and training history visualizations.
A better version of this project can be found in the paper "GallNet An Explainable Deep Learning Model for Multi-Class Gallbladder Disease Classification Using Ultrasound Imaging", which can be accessed by the DOI: 10.21203/rs.3.rs-7625399/v1, https://doi.org/10.21203/rs.3.rs-7812447/v1
